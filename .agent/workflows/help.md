# Workflow: /help

## Available Creative Commands
- **/motion [URL]**: Fetches and implements animations from ReactBits or Aceternity.
- **/perfect """[prompt]"""**: Uses the CO-STAR framework to refine your ideas.
- **/vibe-check**: Researches current design trends via Exa Search.

## Specialized Personas (in agents.md)
- **Creative Director**: Ask for a "3-tier creative brief" (Safe, Modern, Experimental).
- **A11y Inspector**: Ask for an "Accessibility audit" of your components.

## Connected MCP Tools
- **Fetch**: Reading website code.
- **Playwright**: Taking screenshots of `localhost`.
- **Exa**: High-end design & code research.

*Tip: If you're stuck, just ask me: "What should the next step be for my UI?"*
